# sleepysdevin.github.io
A testing ground. Not a portfolio. Just useful things while learning.

## What's Here
- **Dashboard** - Overview of what's available
- **APIs** - Test free, interesting APIs with data fetching
- **Chat** - LLM integration (configure your own API key)
- **Workers** - Cloudflare Workers interaction & proxy testing

## Quick Start
1. Clone: `git clone https://github.com/sleepysdevin/sleepysdevin.github.io.git`
2. Install: `npm install`
3. Dev: `npm run dev`
4. Build: `npm run build`
5. Deploy: `git push origin main` (automatic via GitHub Actions)

## Tech Stack
- React 18
- Vite
- TanStack Query
- Axios
- Dark theme (CSS)
- GitHub Pages
- GitHub Actions (deployment)
